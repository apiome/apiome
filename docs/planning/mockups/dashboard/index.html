Control Panel mockups

Visual designs proposing an enterprise-grade overhaul of the Apiome control panel at /ade/dashboard. Today the control panel is a five-card stat grid plus a recent-activity list. This redesign reorganizes the surface around identity, workspace, specifications, and governance — the four lenses an operator, an owner, and an editor each need — and adds the signals an enterprise customer expects: SSO & SCIM provisioning, RBAC visibility, key rotation timers, IP allowlists, residency badges, audit pulses, lint & quality scores, version-pipeline funnels, sunset countdowns, and live consumer counts on every published version. Visual system reuses the existing Apiome mockup conventions: Inter typography, slate gray scale, panel cards, persisted dark mode, indigo accent.
