Suggested Description

Creates a new product entry in the organization's inventory catalog. The product data is validated against the active ProductSchema, a unique SKU is auto-assigned, and the product is indexed for full-text search within 5 seconds of creation.

The caller must hold the inventory:write OAuth scope. Rate-limited to 500 requests per minute per API key. On success, returns a 201 Created with the full product object.

Suggested Request Body Narrative

Provide a JSON object matching the ProductCreate schema. The name and categoryId fields are required. The optional tags array accepts up to 20 string values.
