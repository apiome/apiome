Apiome Atlas

A JSON Schema 2020-12 type registry stored in a dedicated database. Atlas holds core system types available to every tenant and per-tenant custom types, all addressed by relative $ref URLs rooted at their import source in the API server. The visual editor binds object properties to standard or custom types, and the import system ingests both raw JSON Schemas and type-definition bundles.
