Type Registry — shipping first

The Type Registry is the first governance capability shipping. It stores reusable JSON Schema 2020-12 types in a separate registry database (apiome-types-db), exposes core system types to every tenant, lets tenants define private types, and binds object properties to standard or custom types from the visual editor.
