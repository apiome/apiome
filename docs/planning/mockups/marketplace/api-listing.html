About Commerce API

The Commerce API provides a comprehensive, production-ready platform for building modern e-commerce experiences. It exposes a RESTful interface over 47 endpoints covering the full commerce lifecycle — from product catalog management and dynamic pricing to order processing, inventory tracking, and fulfillment coordination. The API is designed for high-throughput environments, with 99.95% SLA, sub-150ms average latency, and built-in idempotency on all write operations. It supports multi-tenant merchant architectures and integrates natively with the Objectified Identity and Payments Gateway APIs.

What you can do

  • Manage the full product catalog — create, update, version, and archive products with rich metadata and variant support
  • Process orders end-to-end with real-time inventory reservation, tax calculation, and multi-warehouse fulfillment routing
  • Configure dynamic pricing rules, bulk discounts, coupon codes, and promotional campaigns via the Pricing Engine
  • Subscribe to real-time inventory and order events via webhook delivery to build reactive integrations
