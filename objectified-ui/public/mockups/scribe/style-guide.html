Terminology Rules

Terms Scribe will enforce when generating descriptions and guides

Preferred term Avoid / replace Scope Actions
payment intent charge, transaction, order Payments API
API key secret, token, credential All projects
endpoint route, path, URL, method All projects
idempotency key dedup ID, request ID Payments API

Preview Effect

Before

Initiates a charge for the given transaction amount. Provide the secret along with the request to authenticate. The route will return the order details on success.

After

Creates a new payment intent for the specified amount. Authenticate requests using your API key in the Authorization header. The endpoint returns the payment intent details on success.
