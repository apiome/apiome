POST /payments

Create a payment intent

Creates a new payment intent in pending state. The payment must be explicitly captured via POST /payments/{'{id}'}/capture unless capture_immediately is set to true.

curl -X POST https://api.acme.io/v2/payments \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"amount":4999,"currency":"USD"}'
